# Interpretador para código C

## 🪧 Descrição
Interpretador de C como projeto para matéria de Compiladores 1
ministrada pelo professor Sérgio Freitas.

O programa usará um arquivo fonte em C e executará, a partir do retorno do analisador sintático, o conteúdo linha por linha.

## 📝 Progresso
- ❎: Análise Léxica;
- ❎: Análise Sintática;
- ❎: Análise Semântica;
- ❎: Testes;
- 🟩: Documentação;

## ⚠️ Requisitos
- Sistema: Ambiente Unix (Linux, WSL, macOS, etc);
- GCC: Versão 13.3.0 ou superior;
- Flex: Versão 2.6.4 ou superior;
- Bison: Versão 3.8.2 ou superior;
